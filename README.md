# to-doList 

Este é um aplicativo criado com [Expo](https://expo.dev/) e React Native.

npx create-expo-app tot-list --template blank-typescript
npx expo install react-dom react-native-web
npm start


### Pré-requisitos
- [Node.js](https://nodejs.org/) instalado.
- Aplicativo **Expo Go** instalado no seu celular no meu caso (Android ou iOS), ou um emulador configurado no seu computador.

### Passos

1. **Instale as dependências** (caso ainda não tenha feito):
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento**:
   ```bash
   npm start
   ```

3. **Abra o aplicativo**:
   - Pelo celular: Abra o aplicativo **Expo Go** e escaneie o **QR Code** que aparecerá no terminal.
   - Pelo emulador: Pressione `a` no terminal para abrir no Android, ou `i` para abrir no iOS.
   - Pela Web: Pressione `w` no terminal para rodar no navegador.
